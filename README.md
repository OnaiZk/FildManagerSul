## Sobre o Projeto

O **Eletromidia Field Manager - Sul** (Operação Concessão SP) é um sistema web corporativo desenvolvido para gerenciar e otimizar as operações de publicidade Out-Of-Home (OOH) na região Sul de São Paulo. A plataforma foi arquitetada para atender demandas de alta complexidade, oferecendo visualizações precisas em mapas, dashboards estatísticos e painéis de controle em tempo real para as equipes de campo e de gerenciamento estratégico.

Este projeto reflete o desenvolvimento de um produto voltado para o setor empresarial, demonstrando a aplicação de tecnologias modernas para resolver problemas de negócios corporativos, focando em escalabilidade, segurança e alta performance na manipulação de dados operacionais.

## Funcionalidades Principais

- **Mapeamento em Tempo Real:** Integração avançada com Google Maps para monitoramento logístico de frotas, equipes e pontos de publicidade OOH.
- **Dashboards Analíticos:** Gráficos e estatísticas complexas geradas com Recharts para suporte imediato à tomada de decisão gerencial.
- **Interface e Usabilidade:** Design corporativo, moderno e responsivo, focado inteiramente na experiência do usuário (UX) e acessibilidade, desenvolvido utilizando Tailwind CSS.
- **Segurança e Gestão de Dados:** Controle rigoroso de acessos e gerenciamento seguro de estado e informações sensíveis utilizando arquitetura com Supabase.
- **Alta Performance:** Aplicação otimizada construída em React 19 e Vite, assegurando carregamento ágil e estabilidade para operações diárias de alto volume.

## Arquitetura e Tecnologias

- **Frontend:** React 19, TypeScript, Tailwind CSS, Lucide React
- **Roteamento:** React Router DOM
- **Integrações e Visualização:** Recharts, React Google Maps API, ExcelJS para geração de relatórios
- **Backend & Banco de Dados:** Supabase, IndexedDB (estruturação para PWA e suporte offline)
- **Ferramentas de Build:** Vite
- **Testes e Qualidade de Software:** Vitest, React Testing Library
